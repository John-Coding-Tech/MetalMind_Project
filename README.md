# METALMIND PROJECT

AI system to find trusted ACP suppliers and recommend the best value option.

## Overview

This system is designed to:

- Find ACP suppliers in India and China
- Evaluate supplier risk and reliability
- Estimate price from online data
- Recommend the best VALUE supplier

## Key Principle

This is NOT a price comparison tool.

Priority:
1. Trust
2. Risk
3. Reliability
4. Price (secondary)

## System Flow

Tavily → Cleaning → Risk Scoring → Value Score → Ranking → Recommendation → Explanation
